# Backlog — User Story Map

План развития проекта до полноценной современной версии оригинальной Lemmings.
Документ — единственный источник правды по приоритетам: новые задачи добавляются сюда,
сделанные — помечаются, порядок внутри релиза = порядок реализации.

## Definition of Done (для каждой истории)

- [ ] Код соответствует архитектуре проекта (CLAUDE.md): пиксельный террейн, FSM, стратегия навыков.
- [ ] **Автотесты GUT добавлены** для новой логики (регресс + позитивный сценарий).
- [ ] Полный прогон тестов зелёный: `godot --headless -s addons/gut/gut_cmdln.gd -gdir=res://tests -gexit`.
- [ ] Верификатор уровней зелёный: `godot --headless --fixed-fps 2000 -s scripts/verify_levels.gd`.
- [ ] Коммит в формате `type: description`, фиче-ветка, merge в `main` только по команде.

**Правило релиза: перед каждым релизом (merge в `main` и тем более перед сборкой в стор)
весь тестовый сьют и верификатор уровней обязаны проходить.**

| A. Управляю толпой | B. Прохожу кампанию | C. Играю с комфортом | D. Творю и делюсь | E. Получаю продукт |
|---|---|---|---|---|
| назначаю навыки, темп выхода, жертвую | уровни, ранги, прогрессия | реплеи, перемотка, статистика | редактор, обмен уровнями | язык, звук, стор |
