Placeholder. Положите сюда `.ogg` файлы со следующими именами — `AudioManager.play_sfx()` подхватит их автоматически:

- `lets_go.ogg` — спавн леммингов
- `oh_no.ogg` — смерть лемминга
- `yippee.ogg` — лемминг достиг выхода
- `skill_assign.ogg` — назначен навык
- `explosion.ogg` — взрыв bomber'а
- `splat.ogg` — splat при падении

При отсутствии файла AudioManager тихо пропускает воспроизведение.
